# French Haggadah — Haggadah de Pessa'h en Francais

The first open-source, CC0-licensed, structured JSON French Haggadah for both Ashkenazi and Sephardic (Edot HaMizrah) traditions.

Maintained by [SederOS](https://sederos.com) — a free Passover platform for the global Jewish community.

## What's in this repo

| File | Description |
|------|-------------|
| `ashkenazi-fr.json` | Ashkenazi Haggadah — Hebrew + English + French |
| `sephardic-fr.json` | Sephardic / Edot HaMizrah Haggadah — Hebrew + English + French |

Each file contains all sections of the Haggadah with three parallel texts per paragraph:

- **hebrew** — original Hebrew/Aramaic source text
- **english** — CC0 English translation (Sefaria Edition / Sefaria Community Translation)
- **french** — French translation (see below)

## Why this exists

The French-speaking Jewish world is substantial — France alone has the third-largest Jewish population in the world, alongside significant communities in Belgium, Canada, Switzerland, Morocco, Tunisia, Algeria, and their global diaspora.

Despite this, no structured, freely licensed, machine-readable French Haggadah existed anywhere — not on GitHub, not on Sefaria, not anywhere. We built one.

## How to use it

### In a Node.js / JavaScript project

```javascript
import ashkenaziFr from './ashkenazi-fr.json' assert { type: 'json' };
import sephardicFr from './sephardic-fr.json' assert { type: 'json' };

// Get a section in French
function getSection(edition, sectionKey, language = 'french') {
  const data = edition === 'ashkenazi'
    ? ashkenaziFr.sections[sectionKey]
    : sephardicFr.sections[sectionKey];
  return data?.[language] ?? data?.english ?? [];
}

// Example
const dayenuFr = getSection('ashkenazi', 'Magid_Dayenu', 'french');
console.log(dayenuFr[0]);
// -> "S'Il nous avait sortis d'Egypte et n'avait pas execute de jugements..."
```

### Via CDN (jsDelivr)

```
https://cdn.jsdelivr.net/gh/orenplevin/french-haggadah@main/ashkenazi-fr.json
https://cdn.jsdelivr.net/gh/orenplevin/french-haggadah@main/sephardic-fr.json
```

### Direct fetch

```javascript
const res = await fetch(
  'https://cdn.jsdelivr.net/gh/orenplevin/french-haggadah@main/ashkenazi-fr.json'
);
const haggadah = await res.json();
```

## Contributing

Corrections and improvements from native French-speaking Jewish community members are warmly welcome.

If you find a translation that could be improved — especially for key liturgical sections like Dayenu, Ma Nishtana, or Chad Gadya — please open a Pull Request or Issue.

When contributing:

- Focus on liturgical accuracy and natural French flow
- Preserve Hebrew/Aramaic words traditionally kept in the original (e.g. Dayenou, Matzah, Maror, Kadesh)
- Use "L'Eternel" for the divine name, consistent with established French Jewish usage
- Note the tradition (Ashkenazi vs Sephardic) in your PR
