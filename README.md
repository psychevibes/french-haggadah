# French Haggadah / Haggadah en Francais

Complete French translations of the Passover Haggadah, available in two editions:

- **`ashkenazi-fr.json`** — Ashkenazi (standard) Haggadah, 38 sections
- **`sephardic-fr.json`** — Sephardic / Edot HaMizrach Haggadah, 31 sections

Every section includes **Hebrew**, **English**, and **French** text with matching paragraph structure.

## Why this exists

French-speaking Jewish communities (France, Belgium, Canada, Morocco, Tunisia, and diaspora) deserve a high-quality, freely available French Haggadah. This project provides one.

## How it was made

1. English source text fetched from [Sefaria](https://www.sefaria.org) (CC0 / Sefaria Edition)
2. Translated to French by Claude AI with liturgical accuracy guidelines
3. Key sections (Mah Nishtana, Dayenu, Chad Gadya) manually reviewed against established French Jewish sources (Chabad, Torah Box, Harissa)
4. Hebrew liturgical terms preserved in Hebrew script throughout

```json
    }

```

### Sephardic / Edot HaMizrach (31 sections)
Same as Ashkenazi except Nirtzah contains only `Nirtzah_Chad_Gadya` and `Nirtzah_Echad_Mi_Yodea`.

## Usage

```javascript
import ashkenaziFr from './ashkenazi-fr.json';

// Get French text for a section
const dayenu = ashkenaziFr.sections.Magid_Dayenu;
console.log(dayenu.french); // Array of French paragraphs
```

## Contributing

Contributions are welcome! If you find translation errors or want to improve the liturgical accuracy:

1. Fork this repo
2. Edit the relevant JSON file
3. Submit a pull request with a description of what you changed and why

Please preserve:
- Hebrew text in Hebrew script (do not transliterate)
- Standard French liturgical terminology (e.g., "L'Eternel", "accoudés", "Maror")
- Paragraph structure matching the English source

## Translation Notes

- "LORD" / "L-rd" is translated as **"L'Eternel"**
- **"Dayénou"** is preserved alongside the French "cela nous aurait suffi"
- **"Maror"** is kept untranslated (standard in French Jewish usage)
- **"chevreau"** is used for Chad Gadya (not "agneau")
- The Mah Nishtana uses the standard **"Pourquoi cette nuit se différencie-t-elle"**
